# Wild life, todas as vidas importam 

## Contexto

WildLife é uma plataforma web e mobile destinada ao gerenciamento de fauna silvestre, facilitando o registro, monitoramento e acompanhamento de casos relacionados a animais. O sistema visa simplificar o trabalho de profissionais da área ambiental, fornecendo uma interface intuitiva para a gestão de dados e a visualização de estatísticas.

## Funcionalidades

Registro de animais silvestres com informações detalhadas.
Gerenciamento de casos de monitoramento e resgate.
Painel de controle com dados estatísticos, gráficos e rankings de espécies.
Sistema de autenticação e controle de usuários.
Integração com dispositivos móveis para monitoramento em campo.

## Tecnologias Utilizadas
Frontend: HTML5, CSS3, JavaScript.
Template: O design utiliza o tema Mentor disponibilizado pela BootstrapMade.
